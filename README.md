# Konquest
A 2D strategy game 

![img](./media/output.gif)

## Running
You will need Java and GNU Make installed. Then, simply clone the repo, and run:
```
make all
```
Developed using:
- openjdk 24.0.2 2025-07-15
- OpenJDK Runtime Environment Homebrew (build 24.0.2)
- OpenJDK 64-Bit Server VM Homebrew (build 24.0.2, mixed mode, sharing)
- GNU Make 3.81
- MacOS Sequoia 15.5 (24.5.0 Darwin Kernel Version 24.5.0)
